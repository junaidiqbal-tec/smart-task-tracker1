# Smart Task Tracker

A simple task management web application built using HTML, CSS, and JavaScript.

## Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks (All, Active, Completed)

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download the files.
2. Open `index.html` in your browser.
